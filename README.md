# 🏠 House Price Prediction Flask Web App

A Machine Learning web application built with **Flask** that predicts house prices using a trained **Multiple Linear Regression Pipeline**.

---

## 📌 Project Overview

This application allows users to enter house details through a web interface and predicts the estimated house price instantly.

The model was trained using Scikit-learn and integrated into a Flask web application.

---

## 🚀 Features

- Predict house prices
- Responsive user interface
- Flask backend
- Machine Learning Pipeline
- Data preprocessing included in the pipeline
- Professional project structure

## ▶️ Run Locally

Clone the repository:

```bash
git clone https://github.com/junaidramzan-ai-dev/project_1_house_price_prediction_flask_app.git
```

Move into the project:

```bash
cd project_1_house_price_prediction_flask_app
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it.

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser:

```text
http://127.0.0.1:5000
```
