# House Price Prediction

A Flask-based Machine Learning web application that predicts house prices using a trained Multiple Linear Regression Pipeline.

## Features

- House price prediction
- Responsive UI
- Trained Scikit-learn Pipeline
- Flask Backend
- Bootstrap Frontend

## Tech Stack

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- Bootstrap

## Run Locally

```bash
pip install -r requirements.txt

python app.py
```
